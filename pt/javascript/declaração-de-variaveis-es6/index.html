Declaração de variaveis es6

javascript
06 set 17 14h 42m

Na es6 o javascript ganhou mais 2 formas de atribuir variaveis, agora temos 3 tipos de declarações usando:

  1. var = Inicialização opcional, pode ser declarada multiplas vezes com o mesmo nome.
  1. let = Inicialização opcional, faz parte apenas do bloco onde é declarada, não é hosted.
  1. const = Variavel apenas para leitura, não pode declarar uma função ou variável com o mesmo nome no mesmo escopo tambem não é hoisted.
