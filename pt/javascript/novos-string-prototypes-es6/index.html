Na classe string temos 4 novos metodos que podem melhorar a sintax são eles:

  1. repeat = Para repetir a string n vezes.
  1. startsWith = Verifica se a string inicia ou contem o parametro passado em apartir de determinada posicao.
  1. endsWith = Verifica se a string inicia ou contem o parametro passado em apartir de determinada posicao.
  1. includes = Verifica se a string contem o parametro passado em determinada posição.
